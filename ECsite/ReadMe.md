kagoyume

プログラミングキャンプのECサイト構築課題です。

検索結果に出た商品をキープして購入処理を行います。

使用物;

[NetBeans , Java , JSP , Mysql , Bootstrap3 ,Yahoo!API , JacksonAPI ]

仕様;

サイトデザインを統一しています。


ページ上部に「カゴゆめ」のロゴイメージ
その下に各ページに必要なメニューバー
メインページにそのページ特有の処理の表示をしています。

基本的にJSP間ではやり取りせず、Servletを介して処理を行っています。
アクセス制限やセキュリティ、直リンクの禁止処理は行っていません。


概要;

YahooAPIへリクエストを投げ、受け取ったjson形式の情報を分解して情報を格納、その後表示します。
この格納作業にjacksonAPIを使用しています
表示された商品名からは商品詳細にリンクすることができ、リンク先では買い物カゴに商品を保持することができます。
その後保持した商品を買い物かごのページにて購入処理を行います。
買い物かごに入れた商品はこのページから除外することができます。
